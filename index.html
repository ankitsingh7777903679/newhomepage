GSEB 10th Board - Mathematics

Question 1: Solve the equation: 2x + 5 = 15.

Answer: 2x + 5 = 15
2x = 10
x = 5

Question 2: Find the area of a triangle with base 6 cm and height 8 cm.

Answer: Area = (1/2) × 6 × 8 = 24 cm²

BCA Semester 1 - Programming in C

Question 1: Write a C program to print "Hello, World!"

Answer:

#include 
int main() {
  printf("Hello, World!");
  return 0;
}

Question 2: Write a C program to find the sum of two numbers.

Answer:

#include 
int main() {
  int a = 5, b = 10, sum;
  sum = a + b;
  printf("Sum: %d", sum);
  return 0;
}
